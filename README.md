# xrtc_client_js

###Installation:

 git clone -b cima_node_server https://github.com/Comcast/xrtc_client_js.git
 
 sudo npm install

###Directory to save headless client log files
 
 Replace `<directory>` with required directory name in server/controller/authCima.js in line numbers 319 and 324.

###To run the server:

 sudo node server.js
